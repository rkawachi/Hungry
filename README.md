# ChatGPTによるレシピ提案ツール

## 概要
本プロジェクトは、Web UIを通じてChatGPTへプロンプト入力することにより、簡単にレシピを生成できるツールです。

## アーキテクチャ
- **言語**: Python 3.11, JavaScript
- **フレームワーク**: Django
- **API**: OpenAI API
- **モデル**: GPT-4o mini

## ローカルでの動作確認
以下のコマンドを実行するとDjangoサーバが起動します。
```sh
. venv/bin/activate
```
```sh
python3 manage.py runserver
```9
